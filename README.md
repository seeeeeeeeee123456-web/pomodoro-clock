# 🍅 Pomodoro Timer

A cute pastel-themed Pomodoro timer to help you focus and take breaks!

---

## 🌸 Features
- 25 minute study timer
- 5 minute break timer
- Pastel pink, blue and cream theme
- Random motivational quotes
- Animated gradient background

---

## 🛠️ Built With
- HTML
- CSS
- JavaScript

---

## 🚀 How to Use
1. Open the live link above
2. Click **study** to start a 25 min focus session
3. Click **start** to begin the countdown
4. Click **pause** to pause anytime
5. Click **reset** to restart
6. When done, switch to **break** for 5 mins!
